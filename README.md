# Nexcent Landing Page

## Description
A minimal landing page built from a Figma design. This project demonstrates core frontend skills including semantic HTML, responsive layout, and modern CSS techniques.

## Live Demo


## Design
This project is based on a Figma template: https://www.figma.com/design/pe5xcbl4Cv6dP1LezZkv2a/Minimal-Landing-Page-Design-%7C-Website-Home-Page-Design-%7C-Agency-Website-UI-Design--Community-?node-id=0-1&t=hKTyAupsSQIrgSO9-1

## Tech Stack
HTML, CSS3(Flexbox, Grid)

## Features
- Responsive layout (desktop / tablet / mobile)
- Pixel-perfect implementation based on Figma design
- Built using BEM (Block Element Modifier) methodology

## Limitations
- No JavaScript functionality (static layout only)
- Built without preprocessors (no SCSS/SASS)
- No backend integration

## How to Run
Open index.html with Live Server
